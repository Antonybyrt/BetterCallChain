Propagation des blocs — Suivi de chaque bloc produit sur le réseau. [P] = nœud proposeur · [✓] = bloc reçu · [?] = en attente · Spread = délai entre le proposeur et le dernier nœud à recevoir le bloc.
Mempool — Pool de transactions non confirmées de chaque nœud. La barre indique la taille relative du pool (max 10 000 TXs configuré). +N = acceptées · –N = rejetées · évict N = évictions (pool plein, TX de moindre valeur).
Réseau P2P — Graphe des 5 validateurs. Les arêtes = connexions TCP actives maintenues par le connecteur P2P. Pulsation ● = propagation de bloc · ● = gossip TX. Hauteur affichée sous chaque nœud = dernier bloc connu.
