# BetterCallChain

A minimalist blockchain written in Rust using **Proof of Work** consensus.

## Getting Started

```bash
git clone <repo>
cd BetterCallChain
cargo run
```

## Address Format

BetterCallChain addresses use the `bcs1` prefix followed by a base58-encoded string:

```
bcs1qp3wkl...
```
